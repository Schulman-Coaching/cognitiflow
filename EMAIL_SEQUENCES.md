## Email Template Variables

When implementing these sequences, use these merge tags:

- `{first_name}` - Contact's first name
- `{company}` - Company name
- `{industry}` - Industry/vertical
- `{estimated_hours}` - Estimated hours saved based on form input
- `{plan_name}` - Subscription plan name
- `{workflow_count}` - Number of workflows created
- `{task_count}` - Number of tasks automated
- `{time_saved}` - Hours saved during trial
- `{days_remaining}` - Days left in trial
- `{workflow_limit}` - Workflow limit for plan
- `{task_limit}` - Task limit for plan
- `{support_level}` - Support level for plan
